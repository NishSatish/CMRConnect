# CMRConnect
A networking site for a school based in Bangalore, India.

> Setup

* Download or clone.
1.   ``` cd CMRConnect  ```	
		
2.   ``` php artisan key:generate ```
    
    

	
3. To migrate database seeds:
    ```	
    php artisan migrate
    ```

	
4. And finally to start the application, 
    ```		
    php artisan serve
    ```
